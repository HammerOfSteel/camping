## Development & Testing

### Switch Between Themes

Edit `.env`:
```bash
# Option 1
NEXT_PUBLIC_THEME=nordic-standard

# Option 2
NEXT_PUBLIC_THEME=nordic-minimal

# Option 3
NEXT_PUBLIC_THEME=adventure-bold
```

Restart container:
```bash
docker-compose restart frontend
```

Refresh browser at http://localhost:3000 to see the change.

### Test All Three
- Test each theme with:
  - Mobile (375px width)
  - Tablet (768px width)
  - Desktop (1024px+ width)
- Verify buttons, forms, text are readable
- Check hover states
- Ensure accessibility (contrast ratios)

---

## Recommendation for Production

**Nordic Standard** is recommended for the live website because:

1. **Authentic** - Reflects actual Lits brand identity
2. **Timeless** - Won't feel dated in 2-3 years
3. **Accessible** - Best contrast ratios for readability
4. **Versatile** - Works for all seasons and lighting
5. **On-brand** - Captures Ove's peaceful vibe
6. **Market fit** - Appeals to Lits target audience

**Nordic Minimal** can be used for:
- Email marketing campaigns
- Social media graphics
- Premium/luxury positioning emails

**Adventure Bold** can be used for:
- Canoe/adventure trip promotional campaigns
- Social media ads
- Action-focused marketing

## Technical Implementation

All three themes are:
- Fully implemented with Tailwind color scales
- Available in complete 50-900 tone ranges
- Ready for production use
- Accessible (WCAG AA compliant)
- Tested on all screen sizes

### How to Access Theme Colors in Code

```tsx
import { getActiveTheme } from '@/styles/themes'

export default function MyComponent() {
  const theme = getActiveTheme()
  
  // Access colors
  const color = theme.colors.primary.forestGreen  // '#2D5016'
  
  return (
    <div style={{ color: theme.colors.primary.forestGreen }}>
      Themed color!
    </div>
  )
}
```
